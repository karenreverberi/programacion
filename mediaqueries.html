Media Queries

Clase completa + ejercicios

¿Qué son?

Son condiciones en CSS.

👉 Funcionan como un IF: si se cumple → aplico estilos

¿Para qué sirven?

  • Hacer diseño responsive
  • Adaptar a celular, tablet y desktop
  • Mejorar la experiencia del usuario

Sintaxis

@media (max-width: 600px) { body { background: red; } }

max-width vs min-width

max-width (como máximo): aplica hacia abajo

min-width (como mínimo): aplica hacia arriba

Demo

Achicá la pantalla 👇

Cambio de color

Predicción

h1 { color: blue; } @media (min-width: 768px) { h1 { color: red; } }

¿Color en celular? ¿y en desktop?

Respuesta

Celular → azul

Desktop → rojo

Ejercicio 1

Cambiar fondo en pantallas chicas

body { background: blue; } /* completar */

Solución

@media (max-width: 600px) { body { background: red; } }

Ejercicio 2

Mostrar solo en desktop

.mensaje { display: none; } /* completar */

Solución

@media (min-width: 768px) { .mensaje { display: block; } }

Ejercicio 3

Cambiar ancho

.caja { width: 100%; } /* completar */

Solución

@media (min-width: 768px) { .caja { width: 50%; } }

Ejercicio 4

Menú vertical → horizontal

.menu { display: flex; flex-direction: column; } /* completar */

Solución

@media (min-width: 768px) { .menu { flex-direction: row; } }

Ejercicio 5

Grid 1 columna → 3 columnas

.grid { display: grid; grid-template-columns: 1fr; } /* completar */

Solución

@media (min-width: 768px) { .grid { grid-template-columns: 1fr 1fr 1fr; } }

Ejercicio 6

Imagen chica → grande

img { width: 100px; } /* completar */

Solución

@media (min-width: 768px) { img { width: 200px; } }

Ejercicio FINAL (casa)

Hacer tarjetas responsive:

  • Celular → vertical
  • Desktop → horizontal
  • Cambiar color
.contenedor { display: flex; flex-direction: column; } .card { background: lightblue; }

Solución FINAL

@media (min-width: 768px) { .contenedor { flex-direction: row; } .card { background: orange; } }
